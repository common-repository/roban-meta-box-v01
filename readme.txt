Allows admins to create customer fields for any post, page or section.

== Description ==

<p>This Plugin is used for define customer post,page fields,for example,if you want add a select or radio field on your post page,then use it!</p>
<p>这是一个用于自定义post ,page 页面字段的工具,您可以通过这个工具自定义字段</p>
